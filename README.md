# install project
```java
 git clone https://github.com/phannguyendangtruong/upload-cloudinary.git

```

## install maven 
```java
<dependency>
    <groupId>com.cloudinary</groupId>
    <artifactId>cloudinary-http42</artifactId>
    <version>1.1.3</version> 
</dependency>
 ```
